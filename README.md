# Agent Memory

**Remember things for your AI assistant** — save notes, search them later, and keep context across sessions. Works from the terminal or in your browser, backed by [Backboard](https://backboard.io).

```bash
npm install -g @melbourneandrew/agent-memory-cli
```

### A few things you can do

After you [install the CLI](https://melbourneandrew.github.io/agent-memory/installation/) and connect your account:

```bash
agent-memory add "User likes bullet lists and morning standups."
agent-memory search "standups"
agent-memory list
```

Open the local browser UI anytime:

```bash
agent-memory web
```
