# Installation

## Requirements

- Node.js 20 or newer
- npm (or another Node package manager)

Verify Node.js:

```bash
node --version
```

## Install the CLI

```bash
npm install -g agent-memory
```

## Verify Installation

```bash
agent-memory --version
```

Expected output is a semantic version string, for example:

```text
0.1.0
```

## Next Step

After installation, continue to [Configuration](configuration.md) to set your Backboard API key and assistant ID behavior.
